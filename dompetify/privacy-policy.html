Persetujuan Pengguna

Dengan mengunduh, mengakses, atau menggunakan aplikasi Dompetify, Anda menyetujui kebijakan privasi dan ketentuan penggunaan ini. Jika Anda tidak setuju, mohon berhenti menggunakan aplikasi.

1. Privacy Policy

Pengenalan

Dompetify adalah aplikasi pencatatan keuangan keluarga. Aplikasi membantu pengguna mencatat pemasukan, pengeluaran, saldo, kategori, laporan, alokasi, reminder, backup, dan insight keuangan.

Data yang Dikumpulkan

Jenis data yang dapat diproses oleh Dompetify bergantung pada fitur yang Anda gunakan.

  • Data keuangan yang Anda input: transaksi, nominal, kategori, saldo, catatan, tanggal transaksi, alokasi, reminder, dan pengaturan aplikasi.
  • Data penyimpanan lokal: database aplikasi, preferensi tampilan, status onboarding, pengaturan keamanan, dan cache yang diperlukan agar aplikasi berjalan.
  • Data akun Google: jika Anda memilih masuk dengan Google untuk fitur backup atau sinkronisasi, aplikasi dapat memproses nama akun, alamat email, dan token akses yang diberikan oleh Google untuk fitur tersebut.
  • Data backup: jika Anda mengaktifkan backup, data aplikasi dapat disimpan sebagai file backup lokal, di Google Drive app data, atau di server aplikasi untuk backup dan sinkronisasi online.
  • Data pembelian: jika tersedia fitur Pro atau pembelian dalam aplikasi, transaksi diproses oleh Google Play. Alveloper Studios tidak menyimpan detail kartu pembayaran Anda.
  • Data teknis: versi aplikasi, model perangkat, sistem operasi, log error, data performa, atau data diagnostik dapat diproses untuk stabilitas dan dukungan.
  • Data AI insight: jika Anda menggunakan fitur insight berbasis AI, sebagian ringkasan data keuangan dapat dikirim untuk menghasilkan analisis. Hindari memasukkan informasi sensitif di catatan transaksi.
Dompetify dirancang sebagai aplikasi offline first. Data utama yang Anda input tersimpan di perangkat, kecuali ketika Anda memilih fitur yang membutuhkan layanan online seperti login, backup, sinkronisasi, pembelian, update, review, atau AI insight.

Penggunaan Data

  • Menyimpan dan menampilkan catatan keuangan Anda di aplikasi.
  • Menghasilkan laporan, ringkasan, grafik, dan insight keuangan.
  • Menyediakan fitur backup, restore, import, dan export data.
  • Mengaktifkan fitur keamanan seperti PIN, biometrik, dan penyimpanan aman jika tersedia.
  • Memproses pembelian dalam aplikasi dan status fitur Pro melalui Google Play.
  • Meningkatkan kualitas, stabilitas, dan kompatibilitas aplikasi.
  • Memberi dukungan pengguna ketika Anda menghubungi kami.

Penyimpanan dan Backup

Data Dompetify dapat disimpan di perangkat Anda menggunakan database lokal dan penyimpanan aplikasi. Jika Anda membuat backup lokal, file backup berada di lokasi yang Anda pilih atau di folder aplikasi. Jika Anda menggunakan backup online, data dapat dikirim dan disimpan di server aplikasi atau layanan Google sesuai fitur yang Anda aktifkan.

Layanan Pihak Ketiga

Dompetify dapat menggunakan layanan pihak ketiga berikut, sesuai fitur yang aktif pada versi aplikasi Anda:

  • Google Play Services untuk layanan Android, update, review, dan integrasi dasar.
  • Google Sign-In untuk autentikasi akun Google.
  • Google Drive API untuk backup di area penyimpanan aplikasi pengguna.
  • Server aplikasi untuk fitur backup online, sinkronisasi, dukungan backend, atau insight.
  • Google Play Billing untuk pembelian dalam aplikasi.
  • Local notifications untuk reminder di perangkat.

Layanan pihak ketiga memiliki kebijakan privasi masing-masing. Penggunaan layanan tersebut tunduk pada kebijakan dan persyaratan penyedia layanan terkait.

Iklan

Berdasarkan konfigurasi aplikasi saat kebijakan ini dibuat, Dompetify tidak mengaktifkan Google Mobile Ads. Jika iklan ditambahkan pada versi mendatang, kebijakan ini akan diperbarui.

Keamanan Data

Kami menggunakan pendekatan yang wajar untuk menjaga keamanan data, termasuk penyimpanan lokal aplikasi, fitur PIN/biometrik, enkripsi untuk kebutuhan tertentu, dan penyimpanan aman sesuai dukungan perangkat. Namun, tidak ada metode penyimpanan atau transmisi internet yang sepenuhnya bebas risiko.

Hak Pengguna dan Penghapusan Data

  • Anda dapat mengubah atau menghapus data transaksi dari dalam aplikasi.
  • Anda dapat menghapus data lokal dengan menghapus data aplikasi atau uninstall Dompetify dari perangkat.
  • Anda dapat menghapus file backup lokal yang Anda buat.
  • Untuk data yang tersimpan di Google Drive, server aplikasi, atau layanan online lain, Anda dapat menghapus backup dari fitur aplikasi jika tersedia atau menghubungi kami untuk bantuan.
  • Anda dapat mencabut akses akun Google melalui pengaturan akun Google Anda.

Anak-Anak

Dompetify tidak ditujukan secara khusus untuk anak-anak di bawah 13 tahun. Kami tidak secara sadar mengumpulkan informasi pribadi dari anak-anak.

Transfer Data

Jika Anda memakai layanan online seperti Google Drive atau server aplikasi, data dapat diproses di server yang berada di luar negara tempat tinggal Anda sesuai infrastruktur penyedia layanan.

Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini dari waktu ke waktu. Perubahan akan ditampilkan di halaman ini dengan tanggal pembaruan terbaru.
