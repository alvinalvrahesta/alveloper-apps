Dompetify

Aplikasi pencatatan keuangan keluarga untuk mengelola pemasukan, pengeluaran, saldo, kategori, laporan, backup, dan keamanan data personal.

Category: Finance Developer: Alveloper Studios

Overview

Catatan keuangan yang rapi dan pribadi.

Dompetify berfokus pada pencatatan offline terlebih dahulu, lalu memberi opsi fitur tambahan seperti backup, keamanan PIN/biometrik, dan insight keuangan.

Offline first

Transaksi, saldo, kategori, dan pengaturan utama disimpan di database lokal perangkat.

Optional backup

Pengguna dapat memilih backup melalui file lokal atau integrasi Google/Firebase jika fitur tersedia.

Insight ready

Ringkasan dan laporan membantu pengguna memahami kebiasaan keuangan secara lebih cepat.
